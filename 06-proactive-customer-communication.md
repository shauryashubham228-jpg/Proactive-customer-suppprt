# Proactive Customer Communication

## Goal

Inform customers BEFORE frustration escalates.

---

## Communication Triggers

| Scenario | Action |
|---|---|
| Shipment delayed | Send proactive update |
| OFD failed | Reattempt coordination |
| Refund delayed | Status communication |
